## Style Guide

To maintain consistency in our codebase, we follow these basic style guidelines:

### JavaScript:

- Use camelCase for variable names (e.g., userName).

- Use consistent indentation (4 spaces).

- Keep lines under 80 characters.

### Astro:

Follow the official Astro style guide.

### Python:

- Use snake_case for variable names (e.g., user_name).

- Use consistent indentation (4 spaces).

- Keep lines under 79 characters.
